# Project Instructions

## 五位一體協作模式：已啟用

本專案採用五位一體協作模式，詳見 `docs/AI-COLLABORATION-PROTOCOL.md`。

### 快速規則

1. **AI↔AI 溝通**：透過 `docs/` 目錄，不經 LINE
2. **RFC 工作流**：suggestions → RFC → Ready → 通知 Human
3. **LINE 通知**：僅決策型 RFC 即時通知，其餘進每日彙整
4. **Decision Log**：所有決策記錄在 `docs/decision-log.md`
