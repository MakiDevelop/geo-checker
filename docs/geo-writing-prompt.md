# GEO 優化檢查清單（提升 AI 引用潛力）

寫作完成後，請檢查以下項目並給出改善建議：

## 1. 可引用金句（目標：3+ 句）

- 文章中是否有「定義式」句子？（例：「X 是指...」「所謂 X，就是...」）
- 是否有「統計式」句子？（包含具體數字、百分比、時間框架）
- 是否有「觀點式」金句？（簡潔有力、適合被引用的核心論點）

## 2. Schema 結構化資料

- **Article Schema**：確認文章有正確的 Article 或 BlogPosting Schema
- **FAQ Schema**：如果內容適合，建議加入 FAQ 區塊（2-3 個常見問題）
- **Breadcrumb Schema**：確認網站有麵包屑導航

## 3. 實體豐富度（目標：5+ 個命名實體）

- **人名**：提及相關領域的專家、作者、研究者
- **組織**：公司、機構、研究單位
- **專有名詞**：技術名詞、產品名稱、方法論名稱
- 避免過多代名詞（他、她、這個、那個），盡量使用具體名稱

## 4. Q&A 問答結構

- 是否有以問句作為小標題？（例：「什麼是 X？」「如何做到 Y？」）
- 每個問題後是否有明確、直接的回答？
- 建議在文末加入「常見問題 FAQ」區塊

## 5. 外部連結品質（目標：2+ 個）

- 是否引用可信來源？（官方文件、研究報告、權威媒體）
- 連結錨文字是否具描述性？（避免「點這裡」「連結」）
- 內部連結與外部連結是否平衡？

## 6. 首段與代名詞

- 首段是否在 50-200 字內清楚說明文章主題？
- 首段是否包含核心關鍵字和明確定義？
- 段落開頭是否避免使用模糊代名詞（這、那、它、他們）？
